# קורות חיים בקליק

![לוגו קורות חיים בקליק](assets/logo.svg)

מחולל קורות חיים חינמי בעברית, שנבנה כדי לעזור לאנשים לכתוב, לעצב ולייצא קורות חיים בצורה פשוטה ומכבדת.

הפרויקט מיועד להישאר חופשי לשימוש. אין הרשמה, אין שרת, אין Google Analytics, והטיוטה נשמרת בדפדפן המקומי בלבד.

## מה יש כרגע

- עריכת פרטים אישיים, תקציר, ניסיון, השכלה, מיומנויות ושפות
- תצוגה מקדימה בזמן אמת
- כמה תבניות עיצוב וצבעים
- שמירת טיוטה מקומית בדפדפן
- ייצוא ל-PDF, PNG ו-JPEG
- חותמת קרדיט עדינה שאפשר לכבות
- לוגו, favicon ו-manifest בסיסי לפרסום
- שכבת analytics אנונימית, כבויה כברירת מחדל עד להגדרת endpoint

## איך מריצים

פותחים את `index.html` בדפדפן.

אין התקנה ואין build. בשלב הזה האפליקציה נשארת סטטית כדי שיהיה קל לפרסם אותה ב-GitHub Pages ולתרום לה.

## פרסום ב-GitHub Pages

1. מעלים את הפרויקט לריפוזיטורי ציבורי.
2. נכנסים ל-Settings -> Pages.
3. בוחרים Deploy from a branch.
4. בוחרים את `main` ואת תיקיית `/root`.
5. אחרי הפרסום, הקובץ `index.html` יהיה עמוד הבית.

## עקרונות

- תמיד חינם לשימוש.
- פרטיות כברירת מחדל: לא שולחים קורות חיים לשרת.
- מתאים קודם כל לעברית ו-RTL.
- קוד פשוט ונגיש לתורמים מתחילים.
- תמיכה עתידית באנגלית בלי לשבור את החוויה בעברית.

## תמיכה בפרויקט

המטרה אינה להרוויח מהכלי. בעתיד אפשר להוסיף קישור תמיכה אופציונלי למי שרוצה לעזור בתחזוקה, אבל השימוש עצמו יישאר חינמי.
