# Roadmap

## שלב 1 - בסיס לפרסום

- `index.html` כנקודת כניסה ל-GitHub Pages
- README ברור בעברית
- רישיון פתוח
- מסמך תרומה
- מיתוג בסיסי: לוגו SVG, favicon וחותמת עדינה

## שלב 2 - שיפור חוויית כתיבה

- טיפים לפי שדה: תקציר, ניסיון, הישגים ומיומנויות
- דוגמאות נוספות לפי מקצועות
- בדיקות תוכן בסיסיות: אימייל, טלפון, קישורים, שדות חסרים
- מצב "מדריך" למשתמשים שכותבים קורות חיים בפעם הראשונה

## שלב 3 - ארכיטקטורה נקייה יותר

- פיצול הקובץ ל-`src/app.js`, `src/styles.css` ותבניות
- מעבר הדרגתי ל-build קל רק אם זה באמת מוסיף ערך
- בדיקות ייצוא PDF ותצוגה במובייל
- תיעוד מבנה נתוני CV כדי להקל על תרומות

## שלב 4 - שפות ותבניות

- תמיכה באנגלית ו-LTR
- החלפת שפה לכל ממשק האפליקציה
- תבניות מותאמות לענפים שונים
- ייבוא/ייצוא JSON לטיוטות

## שלב 5 - קהילה ותמיכה

- GitHub Issues מסודרים לבאגים ורעיונות
- תבניות PR
- דף "איך לכתוב קורות חיים טובים"
- קישור תמיכה אופציונלי, בלי paywall ובלי נעילת פיצ'רים
