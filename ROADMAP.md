# Roadmap

## שלב 1 - בסיס לפרסום ✅

- `index.html` כנקודת כניסה ל-GitHub Pages
- README ברור בעברית
- רישיון פתוח
- מסמך תרומה
- מיתוג בסיסי: לוגו SVG, favicon וחותמת עדינה

## שלב 2 - ארכיטקטורה מודולרית ✅

- פיצול הקוד ל-ES modules: `src/app.js`, תבניות, ייצוא, אחסון
- תבניות CV כפונקציות render נפרדות עם קומפוננטות משותפות
- מערכת סקציות גנרית: רישום, הוספה, הסרה, drag & drop
- סקציות חדשות מובנות: שירות צבאי, התנדבות, פרויקטים, הסמכות
- ייבוא/ייצוא JSON
- תשתית i18n
- טיפים לכתיבה ובדיקות תוכן
- דוגמאות בקבצי JSON נפרדים

## שלב 3 - שיפור חוויית כתיבה

- טיפים מתקדמים לפי שדה: תקציר, ניסיון, הישגים ומיומנויות
- דוגמאות נוספות לפי מקצועות
- מצב "מדריך" למשתמשים שכותבים קורות חיים בפעם הראשונה
- שיפור נגישות: ARIA, focus management, ניווט מקלדת

## שלב 4 - שפות ותבניות

- תמיכה באנגלית ו-LTR
- החלפת שפה לכל ממשק האפליקציה
- תבניות מותאמות לענפים שונים
- תבניות נוספות

## שלב 5 - ייצוא מתקדם

- PDF עם טקסט ניתן לבחירה (ללא html2canvas) — pdfmake או דומה
- ייצוא ל-DOCX
- שיתוף קישור זמני (אופציונלי)

## שלב 6 - קהילה ותמיכה

- GitHub Issues מסודרים לבאגים ורעיונות
- דף "איך לכתוב קורות חיים טובים"
- קישור תמיכה אופציונלי, בלי paywall ובלי נעילת פיצ'רים
- אפשרות build עם Vite לביצועים (אופציונלי)
